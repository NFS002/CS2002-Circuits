13 /* This number (1000000) was chosen as the runtime upper limit because computing truth tables and timeseries for any longer times
14  would signifiicantly reduce performance. */

16 /*If every node in the circuit was a wire, given the runtime limit, there could be a maximum of
17  Log2(max_circuit_time) = Log10(max_circuit_time)/Log10(2) nodes in the circuit, so therfore this number (20) was chosen as the maximum circut size
18  C99 does not have a log2 function, so we must convrert to log10. I have chosen to precompute
19  this number, and round down to the closest integer, so as to improve performance of the program, rather than computing it every time it is used. */
